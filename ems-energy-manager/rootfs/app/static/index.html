📡 SolarEdge Modbus TCP

Modbus TCP aanzetten: SolarEdge SetApp → Communicatie → LAN → Modbus TCP inschakelen. Poort is meestal 1502 (nieuwe firmware) of 502 (oud).

🔋 Sessy lokale REST API

Gebruikersnaam en wachtwoord staan op de sticker op de Sessy dongle. De lokale API werkt zonder cloud.

🔌 Easee Cloud API

Easee heeft geen lokale API. De EMS logt in op je Easee account en beheert de laadpaal via de cloud. Token wordt automatisch ververst.

📊 HomeWizard lokale API

Lokale API aanzetten in de HomeWizard Energy app: Instellingen → Meter → Wi-Fi P1 → Lokale API inschakelen.
